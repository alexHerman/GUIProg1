Weather Service Readme
This program was created using the NetBeans IDE 8.1.
Authors: Alex Herman, Austin Rotert, Evan Hammer.

Compilation:
	After opening the project in the NetBeans IDE. Run the project either
	by clicking on the rightword facing arrow, or by running from the run menu.
Statistics:
	Statitistics can be seen on the right side panel.  This will display statistics
	for the range of values displayed on the graph.  To change between the different
	ranges, just click on one of the radio buttons and you can adjust the time using
	the spinner above the graph.  However to make the spinner update the graph, just hit
	Go.
